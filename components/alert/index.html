Alerte

Les alertes permettent d’attirer l’attention de l’utilisateur sur une information sans interrompre sa tâche en cours.

Alerte de succès avec du contenu

Une alerte a un type obligatoire parmi :error, :success, :info, :warning.

Le titre title est aussi obligatoire

Alerte d’erreur taille SM sans contenu

La taille size peut être :sm ou :md (le défaut).

En taille SM le contenu n’est pas obligatoire

Alerte avec un bouton pour fermer
